### 项目简介

此项目使用深度学习方法，专门设计来识别残缺的文字。基于Python，Pytorch, OpenCV等工具和库进行开发，主要利用了深度学习和注意力机制来加强模型的效果。项目获得了软件著作权1项。

### 主要技术和工具

- **语言和框架**:
  - Python
  - Pytorch
- **图像处理**:
  - OpenCV
- **核心技术**:
  - 深度学习
  - 注意力机制

### 主要特性

1. **数据增强**:
    - 使用**图形学增强**和**CutMix**对2000+中文常用字进行增强，生成大量训练样本。
  
2. **形态学操作增强**:
    - 利用**Erosion**和**Dilation**等形态学操作来强化文字的轮廓，从而提高模型的准确率。
  
3. **注意力机制**:
    - 采用注意力机制让模型更加集中于残缺文字中特征密集的部分，从而提高识别的准确性。
  
4. **出色的准确率**:
    - 即使在30%的文字遮挡情况下，识别准确率也能达到69.32%，这是通用识别模型（例如：阿里云OCR）准确率的11.32倍。
